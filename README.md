# Managed Ecosystem Microbial Ecology Lab Website

This repository hosts the public website for the MEME Lab (Purdue University).

## Tech stack
- Vite
- React
- Tailwind CSS
- GitHub Pages (static hosting)

## Content management
Public content is maintained via:
- Google Forms → Google Sheets → Apps Script (JSON endpoints)
- Third-party platforms (Zotero, protocols.io, Google Photos, OSF)
